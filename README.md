# Taita Craft & Cultural Heritage Centre (TCCHC) Website

A modern, culturally-rich website for the Taita Craft & Cultural Heritage Centre in Taita, Kenya. This website showcases TCCHC's mission to preserve Dawida heritage, foster artisan-consumer connections, and promote sustainable livelihoods.

## About TCCHC

Founded in 2026 by Dr. Mjomba Majalia, TCCHC is a sustainable, eco-friendly space dedicated to:
- Preserving Dawida (Taita) heritage and traditional knowledge
- Fostering connections between artisans, designers, and consumers
- Creating sustainable livelihoods for craft communities and Mwazindika spiritual dance artist groups
- Promoting rural-urban links through cultural exchange
- Revitalization, documentation, archiving, and preservation of Dawida tangible and intangible cultural heritage
- Hosting the annual Taita Craft Week and International Mwazindika Drumming Festival

## Vision
To be a leading crafts and cultural heritage center in innovations, revitalization, documentation and preservation of Dawida traditional knowledge.

## Mission
TCCHC acts as a bridge between artisans, designers, and consumers to create sustainable livelihoods for craft communities and Mwazindika spiritual dance artist groups, directly connecting them to global and national markets.

## Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Cultural Authenticity**: Design inspired by Kenyan/Taita culture with earth-tone color palette
- **Accessibility**: WCAG compliant with semantic HTML and ARIA labels
- **Modern Stack**: Built with React, TypeScript, and Tailwind CSS

## Website Sections

- **Home**: Overview of TCCHC's mission and featured programs
- **About TCCHC**: Detailed information about our vision and history
- **Craft Programs**: Workshops in metal work, pottery, weaving, tie-dye, and Mwazindika dance
- **Events & Festivals**: Taita Craft Week and International Mwazindika Drumming Festival
- **Artist Residency**: Programs for local and international artists
- **Gallery & Shop**: Exhibition space and artisan marketplace
- **Heritage Archive**: Digital and physical preservation of Dawida knowledge
- **Visit Us**: Location information and visiting hours
- **Cultural Blog**: Stories and updates from our community
- **Support TCCHC**: Donation and partnership opportunities
- **Contact**: Get in touch with our team

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with custom cultural color palette
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```
